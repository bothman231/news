About.html

This app is a HTML-5 Js client, using the following server technologies

Spring Framework 4.3.11.RELEASE For the core product
Spring Boot 1.5.6.RELEASE As a quick run
Apache Maven 3.5.0 For dependency management
STS - Spring source tool built on eclipse 2.9.0 For an IDE
Pivitol TC Server build into STS v3.0,v3.2 For a server
Hibernate 5.0.12.Final For persistence
Git & Github For Source code control
Microsoft MS SQL 13.0.4001 As a Relational Database
Java 1.8 As a coding language
Jenkins 2.73.1 As a Continuous Integration server

JPA is a specification
Hibernate is an implementation of that specification.
